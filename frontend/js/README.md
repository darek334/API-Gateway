# Skrypty JS Sterujące Interakcją w Przeglądarce
Folder zawierający główne pliki JavaScript.
Są to skrypty sterujące i obsługujące wszystko to co dzieje się na stronie.
Jest to główny silnik formularza po stronie użytkownika.
Wszystko to odpowiada za graficzną interakcję z użytkownikiem poprzez zdarzenia w przeglądarce internetowej.
