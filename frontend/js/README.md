# Skrypty JS sterujące interakcją w przeglądarce
Folder zawiera główne pliki JavaScript wykonywane po stronie przeglądarki internetowej.
Skrypty odpowiadają za obsługę interfejsu użytkownika, dynamiczne zmiany wyglądu strony, zbieranie danych wprowadzanych przez użytkownika, oraz ich wstępne przetwarzanie i formatowanie.
Przygotowane dane są następnie wysyłane do odpowiednich podstron, gdzie po stronie serwera są przetwarzane przez skrypty PHP, wykonujące właściwe zapytania HTTP z wykorzystaniem modułu cURL.
